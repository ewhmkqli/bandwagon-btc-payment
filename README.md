## 那国内用户怎么付款最省事？

**支付宝是最顺的路。**

整个流程大概这样：选好套餐，加入购物车，填写账号信息时选 Alipay，点 Complete Order，然后直接跳出支付宝二维码，扫一下就完成了。前前后后五分钟以内。

有几个要注意的地方：

1. 注册账号时不要挂代理——搬瓦工会检测，有时候会被判定为欺诈订单直接取消
2. 账号信息如实填写，邮箱要真实，收款确认和后续通知都走这里
3. 支付宝付款是人民币结算，按当天汇率换算成美元——汇率正常情况下没啥损失

银联也可以，流程差不多。如果你支付宝临时有问题，银联是个备选。

## 账户充值同样不支持加密货币

顺带提一句：搬瓦工的账户余额充值（Add funds to account）只支持 PayPal 和信用卡，不支持支付宝、银联，更没有比特币。

充值余额的实际用途是方便一次充入多个月的费用，让续费自动扣款——账单出来时后台直接扣余额，不用每次手动付。搬瓦工官方说明是不会自动从你绑定的支付方式扣款，需要你手动或者提前充好余额。

如果你计划长期用搬瓦工，可以考虑充入 2-3 个续费周期的金额，省得每次续费时还要手动操作一遍。

## 常见问题

**Q：搬瓦工之前支持过比特币吗？**

有老文章提到过，某些地区曾经可以选到 BTC 支付。但这个选项目前的官方结账页面上找不到，不管哪个地区都一样——实际购买时不要抱期望。

**Q：想匿名购买搬瓦工 VPS，有什么办法？**

搬瓦工本身不提供完全匿名的购买选项。加密货币支付也没有，目前可用的最接近的方式是 PayPal——但 PayPal 本身需要实名，所以也谈不上真正匿名。如果这是需求，建议考虑其他明确支持比特币的 VPS 商家。

**Q：支付宝付款安全吗？会不会被风控？**

买搬瓦工走支付宝是正常的外汇消费，就是跨境购物付款，没有什么特别的问题。唯一要注意的是注册时不要挂代理，不然会被判定为欺诈，订单会被取消（自动退款）。

**Q：搬瓦工续费支持支付宝吗？**

支持。续费页面和首购一样，选 Alipay 就行。搬瓦工会提前 7 天发续费账单邮件，点进去按和首购一样的流程操作就可以了。

**Q：没有支付宝和信用卡怎么买搬瓦工？**

有 PayPal 账号的话最方便。或者银联也行，大部分国内银行卡都支持。如果这几样都没有，搬瓦工就真的不太好买了——它不接加密货币。
